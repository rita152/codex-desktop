# Docs (docs/)

## Overview

Project documentation and AI engineering assets.

## Structure

- `ai/`: AI engineering assets
  - `TODO_NOW.md`: active tasks and status tracking
  - `CHANGELOG_DEV.md`: development changelog (recent changes)
  - `PROMPT_ENHANCE_PLAN.md`: feature implementation plan (completed)
- `screen_shoot.png`: application screenshot

## Conventions

For any new feature docs:
- Title should clearly state it describes the **current implementation**
- Include: prerequisites, workflow steps, troubleshooting, implementation pointers
